## 案例20：智能推荐系统候选商品筛选
**业务场景**：电商平台需要基于用户行为数据，为每个用户生成个性化商品推荐列表。  
**SQL书写逻辑**：
1. 统计用户30天内的商品交互行为
2. 找出有相似购买行为的用户
3. 计算推荐分数(相似用户购买次数加权)
4. 排除用户已交互过的商品
5. 按推荐分数降序输出结果

```sql
WITH user_behavior AS (
    SELECT 
        user_id,
        product_id,
        SUM(CASE WHEN behavior_type = 'view' THEN 1 ELSE 0 END) AS view_count,
        SUM(CASE WHEN behavior_type = 'cart' THEN 1 ELSE 0 END) AS cart_count,
        SUM(CASE WHEN behavior_type = 'purchase' THEN 1 ELSE 0 END) AS purchase_count,
        MAX(event_time) AS last_interaction_time
    FROM user_behaviors
    WHERE event_time >= DATE_SUB(CURRENT_DATE, INTERVAL 30 DAY)
    GROUP BY user_id, product_id
),
similar_users AS (
    SELECT 
        u1.user_id AS target_user,
        u2.user_id AS similar_user,
        COUNT(DISTINCT u2.product_id) AS common_products,
        ROW_NUMBER() OVER (PARTITION BY u1.user_id ORDER BY COUNT(DISTINCT u2.product_id) DESC) AS similarity_rank
    FROM user_behavior u1
    JOIN user_behavior u2 ON u1.product_id = u2.product_id AND u1.user_id != u2.user_id
    WHERE u1.purchase_count > 0 AND u2.purchase_count > 0
    GROUP BY u1.user_id, u2.user_id
    HAVING COUNT(DISTINCT u2.product_id) >= 3
),
recommendation_candidates AS (
    SELECT 
        s.target_user,
        b.product_id,
        p.product_name,
        p.category,
        SUM(b.purchase_count * CASE WHEN s.similarity_rank <= 10 THEN 1 ELSE 0.5 END) AS recommendation_score,
        MAX(b.last_interaction_time) AS last_interaction
    FROM similar_users s
    JOIN user_behavior b ON s.similar_user = b.user_id
    JOIN products p ON b.product_id = p.product_id
    LEFT JOIN user_behavior ub ON s.target_user = ub.user_id AND b.product_id = ub.product_id
    WHERE ub.product_id IS NULL  -- 排除用户已经交互过的商品
    AND s.similarity_rank <= 20  -- 只考虑前20个相似用户
    GROUP BY s.target_user, b.product_id, p.product_name, p.category
)
SELECT 
    target_user,
    product_id,
    product_name,
    category,
    recommendation_score
FROM recommendation_candidates
ORDER BY target_user, recommendation_score DESC
LIMIT 1000;
```
