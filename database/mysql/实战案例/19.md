## 案例19：计算客户生命周期价值（LTV）
**业务场景**：电商企业需要分析不同时期获取的客户价值，评估获客成本合理性。  
**SQL书写逻辑**：
1. 找出每个客户的首次购买日期
2. 计算客户获取后的月数
3. 按3个月为周期分组统计
4. 计算每组的平均收入、订单数和客户数
5. 推算年化LTV

```sql
WITH first_purchases AS (
    SELECT 
        customer_id,
        MIN(order_date) AS first_order_date
    FROM orders
    GROUP BY customer_id
),
customer_months AS (
    SELECT 
        customer_id,
        TIMESTAMPDIFF(MONTH, first_order_date, CURRENT_DATE) AS months_since_first_order
    FROM first_purchases
    WHERE first_order_date >= DATE_SUB(CURRENT_DATE, INTERVAL 3 YEAR)
),
customer_revenues AS (
    SELECT 
        o.customer_id,
        c.months_since_first_order,
        SUM(o.order_amount) AS total_revenue,
        COUNT(DISTINCT o.order_id) AS order_count
    FROM orders o
    JOIN customer_months c ON o.customer_id = c.customer_id
    GROUP BY o.customer_id, c.months_since_first_order
),
cohort_analysis AS (
    SELECT 
        FLOOR(months_since_first_order / 3) * 3 AS cohort_month,
        AVG(total_revenue) AS avg_revenue,
        AVG(order_count) AS avg_orders,
        COUNT(DISTINCT customer_id) AS customer_count
    FROM customer_revenues
    GROUP BY FLOOR(months_since_first_order / 3) * 3
)
SELECT 
    cohort_month,
    cohort_month + 3 AS next_cohort,
    customer_count,
    avg_revenue,
    avg_orders,
    avg_revenue / NULLIF(avg_orders, 0) AS avg_order_value,
    avg_revenue * 12 / (cohort_month + 1) AS projected_annual_ltv
FROM cohort_analysis
ORDER BY cohort_month;
```
